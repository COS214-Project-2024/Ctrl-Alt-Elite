\doxysubsection{Detailed Description}
Abstract base class representing a resource. 

The \mbox{\hyperlink{classResource}{Resource}} class serves as an abstract base for different types of resources such as \mbox{\hyperlink{classWater}{Water}}, \mbox{\hyperlink{classEnergy}{Energy}}, and \mbox{\hyperlink{classMaterials}{Materials}}. It defines common properties and methods that all resources must implement, such as capacity management and cost retrieval.
