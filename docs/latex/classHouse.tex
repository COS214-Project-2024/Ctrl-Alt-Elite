\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classHouse}{House}} class represents a residential structure within a city. 

This class inherits from the \mbox{\hyperlink{classResidential}{Residential}} class and provides specific implementations for handling house-\/related operations such as displaying details, calculating occupancy, and managing maintenance.
