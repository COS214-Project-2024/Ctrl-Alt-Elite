\doxysubsection{Detailed Description}
Represents an apartment building in the city simulation. 

The \mbox{\hyperlink{classApartment}{Apartment}} class, derived from \mbox{\hyperlink{classResidential}{Residential}}, models an apartment building, providing functions to display the apartment\textquotesingle{}s information, calculate occupancy, and perform maintenance.
