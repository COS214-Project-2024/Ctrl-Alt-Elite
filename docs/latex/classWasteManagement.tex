\doxysubsection{Detailed Description}
Manages waste collection and recycling processes in a building. 

The \mbox{\hyperlink{classWasteManagement}{Waste\+Management}} class inherits from the \mbox{\hyperlink{classUtilities}{Utilities}} class and provides functionality to handle waste management tasks, including applying utility to buildings, checking waste capacity, and increasing recycling rates.
