\doxysubsection{Detailed Description}
Represents a monument, which is a specific type of landmark. 

The \mbox{\hyperlink{classMonument}{Monument}} class extends the \mbox{\hyperlink{classLandmarks}{Landmarks}} class, providing specific functionality and characteristics associated with monuments, including how they are displayed, how their entertainment rating is updated, and how maintenance is performed.
