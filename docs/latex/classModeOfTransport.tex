\doxysubsection{Detailed Description}
Abstract class representing a mode of transport. 

This class extends the \mbox{\hyperlink{classTransportStrategy}{Transport\+Strategy}} interface and manages the current state of transportation. It utilizes the State design pattern to handle various transportation modes.
