\doxysubsection{Detailed Description}
Represents a taxi service in the public transport system. 

The \mbox{\hyperlink{classTaxi}{Taxi}} class extends the \mbox{\hyperlink{classPublicTransport}{Public\+Transport}} class to provide functionality specific to taxi services, including fare management and travel logic.
