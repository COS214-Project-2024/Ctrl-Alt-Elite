\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classIndustrial}{Industrial}} class represents an industrial building in a city. 

This class inherits from the \mbox{\hyperlink{classBuilding}{Building}} class and includes specific attributes and methods for managing employment ratings and pollution levels associated with industrial buildings.
