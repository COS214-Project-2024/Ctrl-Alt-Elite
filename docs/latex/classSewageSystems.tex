\doxysubsection{Detailed Description}
Class representing sewage systems within a city building. 

The \mbox{\hyperlink{classSewageSystems}{Sewage\+Systems}} class manages the sewage generation and capacity for a specific building. It is derived from the \mbox{\hyperlink{classUtilities}{Utilities}} class.
