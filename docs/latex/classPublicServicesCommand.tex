\doxysubsection{Detailed Description}
\mbox{\hyperlink{classCommand}{Command}} class for managing public services within a city. 

This class encapsulates the command to update and manage public services provided by the \mbox{\hyperlink{classPublicServicesDepartment}{Public\+Services\+Department}}. It holds the necessary data to execute the command and interacts with the public services department.
