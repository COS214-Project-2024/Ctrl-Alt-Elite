\doxysubsection{Detailed Description}
Factory class for managing and providing resources. 

The \mbox{\hyperlink{classResourceFactory}{Resource\+Factory}} class creates and maintains a pool of resources such as \mbox{\hyperlink{classEnergy}{Energy}}, \mbox{\hyperlink{classWater}{Water}}, \mbox{\hyperlink{classMaterials}{Materials}}, and \mbox{\hyperlink{classBudget}{Budget}}. It provides methods for resource retrieval, status display, and replenishment.
