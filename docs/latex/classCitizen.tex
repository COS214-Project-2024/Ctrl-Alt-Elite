\doxysubsection{Detailed Description}
Represents a citizen with preferences, satisfaction level, and interactions with public services. 

The \mbox{\hyperlink{classCitizen}{Citizen}} class includes attributes related to transportation preferences, state, and personal details, and provides functionality for managing transport, satisfaction, and service interactions.
