\doxysubsection{Detailed Description}
Manages the financial resources allocated for various city functions. 

The \mbox{\hyperlink{classBudget}{Budget}} class, inheriting from \mbox{\hyperlink{classResource}{Resource}}, tracks the city\textquotesingle{}s budget, allowing for consumption and replenishment. It also allows querying the available budget.
