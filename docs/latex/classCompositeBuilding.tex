\doxysubsection{Detailed Description}
Represents a composite building that can contain multiple buildings. 

The \mbox{\hyperlink{classCompositeBuilding}{Composite\+Building}} class aggregates multiple \mbox{\hyperlink{classBuilding}{Building}} objects and manages overall attributes like population and jobs. It provides functionalities for adding, removing, and displaying individual buildings, as well as managing city-\/wide metrics.
