\doxysubsection{Detailed Description}
Represents a bike in the transportation system. 

The \mbox{\hyperlink{classBike}{Bike}} class inherits from \mbox{\hyperlink{classPrivateTransport}{Private\+Transport}}, allowing it to represent a personal transportation option in the simulation. It supports a state of availability and can be used for travel when available.
