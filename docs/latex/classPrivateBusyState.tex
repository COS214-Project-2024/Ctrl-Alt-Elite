\doxysubsection{Detailed Description}
Represents the busy state of a private transport mode. 

The \mbox{\hyperlink{classPrivateBusyState}{Private\+Busy\+State}} class indicates that the private transport is currently in use and unable to accept new requests. It handles state transitions and manages behavior specific to the busy state.
