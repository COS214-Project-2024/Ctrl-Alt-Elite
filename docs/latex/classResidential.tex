\doxysubsection{Detailed Description}
Represents a residential building with occupancy management. 

The \mbox{\hyperlink{classResidential}{Residential}} class is derived from the \mbox{\hyperlink{classBuilding}{Building}} class and includes functionality to manage the number of residents, available units, and occupancy rates within a residential building. It provides methods for displaying information, calculating occupancy, and performing maintenance.
