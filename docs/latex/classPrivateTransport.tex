Represents a private mode of transport with associated functionalities. 

The \mbox{\hyperlink{classPrivateTransport}{Private\+Transport}} class extends the \mbox{\hyperlink{classModeOfTransport}{Mode\+Of\+Transport}} class, providing implementations for travel, availability checks, and commute time calculations.
