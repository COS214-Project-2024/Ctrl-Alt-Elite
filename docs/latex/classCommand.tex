\doxysubsection{Detailed Description}
Interface for command pattern, representing an executable action. 

The \mbox{\hyperlink{classCommand}{Command}} interface defines a standard structure for encapsulating actions as objects, enabling parameterization of clients with queues, requests, and operations.
