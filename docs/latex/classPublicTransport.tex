\doxysubsection{Detailed Description}
Represents a public mode of transportation. 

This class inherits from the \mbox{\hyperlink{classModeOfTransport}{Mode\+Of\+Transport}} class and encapsulates the details of public transport such as capacity, available routes, and functionalities to travel and stop at stations.
