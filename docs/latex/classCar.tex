Represents a private transport vehicle (car) that uses petrol for travel. 

The \mbox{\hyperlink{classCar}{Car}} class extends \mbox{\hyperlink{classPrivateTransport}{Private\+Transport}} and provides a specific implementation for the travel method and petrol-\/related functionality.
