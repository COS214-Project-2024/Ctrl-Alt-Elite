\doxysubsection{Detailed Description}
Represents the busy state of a public transport mode. 

The \mbox{\hyperlink{classBusyState}{Busy\+State}} class inherits from \mbox{\hyperlink{classPublicTransportState}{Public\+Transport\+State}} and handles the behavior of a public transport mode when it is busy and cannot accept more passengers.
