\doxysubsection{Detailed Description}
Represents a water resource with capacity management. 

The \mbox{\hyperlink{classWater}{Water}} class inherits from the \mbox{\hyperlink{classResource}{Resource}} class and provides functionality to manage water resources, including consumption and replenishment of water supplies.
