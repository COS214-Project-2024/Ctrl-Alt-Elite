\doxysubsection{Detailed Description}
Represents a utility for generating and supplying electricity to buildings. 

The \mbox{\hyperlink{classPowerPlants}{Power\+Plants}} class is responsible for managing power generation and consumption, ensuring that buildings receive the necessary electricity for operation.
