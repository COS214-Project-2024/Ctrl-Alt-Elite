Detailed Description

Represents the public services department responsible for managing community services and infrastructure.

Forward declaration of the Citizen class.

Represents the public services management within a city.

This class inherits from the Government class and is responsible for managing public services in the city. It provides methods to update available services, manage the city, and handle requests related to public services.
